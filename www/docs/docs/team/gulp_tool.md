## Start

 - Check out or download this repository
 - Enter your local directory, and install dependencies:

``` bash
cnpm i
```

## 已添加的工具及对应命令

### svgAddClass
  - svg文件批量添加以parseSVG-{文件名}命名的唯一标示的Class,运行命令
  - 用于处理HH70上UE新提供的SVG图片

``` bash
gulp svgAddClass
```
